邁克爾·肯普·蒂皮特爵士，OM，CH，CBE（英語：Sir Michael Kemp Tippett，1905年1月2日－1998年1月8日），英國20世紀著名作曲家。

# 生平
```
蒂皮特出生於倫敦，上中學時才開始接觸音樂。
1923年-1928年在英國皇家音樂學院學習。
早期在第一次世界大戰、大蕭條等社會背景下深受左翼政治影響。
1940年至1951年擔任倫敦莫利學院（Morley College）的音樂主任。
1943年因反對服兵役被判入獄3個月，他終生堅持和平主義的立場。

此後，他把主要精力放在作曲上，於1945年創作了第一部交響曲後，
開始寫作第一部歌劇《仲夏結婚》。這些早期作品對他後來的創作有很大影響。
60年代開始，他的作品通過錄音被廣泛傳播。
1966年封為爵士。1983年獲功績勳章。
1997年，他在去瑞典斯德哥爾摩主持作品回顧音樂會期間患上肺炎，回到倫敦的家中後於次年初去世。
```
